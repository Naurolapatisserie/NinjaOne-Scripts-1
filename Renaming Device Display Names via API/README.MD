Ce script suppose que vous avez un CSV avec deux colonnes, une avec un en-tête systemName, et une colonne avec un en-tête displayName. Insérez le chemin de votre fichier CSV à la ligne 40 du script joint. Le script tente ensuite de faire correspondre l'appareil dans Ninja basé sur ce systemName. Cela suppose également que toutes vos machines ont des noms système uniques - si vous avez des noms système en double, vous devrez trouver un autre critère de correspondance. Cette version du script peut très facilement être modifiée pour utiliser directement l'ID d'appareil unique dans Ninja au lieu de la correspondance de nom basée sur le nom système.

Vous devrez créer des identifiants API en utilisant les credentials client avec les portées monitoring et management, en utilisant le type d'application machine-to-machine.
